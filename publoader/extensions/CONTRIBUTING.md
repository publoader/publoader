# Contributing extensions

Extensions live in their own repos and are pulled into this directory at
runtime — do not commit extension code here.

- **Public extensions** → [publoader-extensions](https://github.com/publoader/publoader-extensions)
- **Private / vendor-supplied extensions** → `publoader-extensions-private` (invite-only)

For the full contributing guide (manifest schema, public API surface,
required dataclasses, scheduling, AST safety scan, submission flow),
see the [extensions repo README](https://github.com/publoader/publoader-extensions/blob/master/README.md).
